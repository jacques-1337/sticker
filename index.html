32er Worldwide 🌍

Stahnsdorf 32er Sticker auf der ganzen Welt

🎯 Was ist das?

Die 32er Community aus Stahnsdorf klebt ihre Sticker weltweit – und diese App macht alle Spots auf einer Karte sichtbar. Je weiter der Sticker von Stahnsdorf weg ist, desto mehr Punkte gibt's!

📍 So funktioniert's

  • Stadt suchen, wo du den Sticker gesetzt hast
  • Optional: Foto hinzufügen (+5 Punkte)
  • Auf der Karte erscheint dein Spot für alle sichtbar
  • Community-Score zeigt alle Punkte zusammen

🔒 Datenschutz

  • Kein Account, kein Name erforderlich
  • GPS wird auf ~1 km Radius verschleiert
  • EXIF-Metadaten werden aus Fotos entfernt
  • Keine persönlichen Daten werden gespeichert

⚙️ Technik

  • Karte: OpenStreetMap + Leaflet.js
  • Datenbank: Supabase (Frankfurt)
  • Hosting: Cloudflare Pages
  • Domain: 32-er.com
  • Kosten: 0 €/Monat
